# TodoListExercise

## 사용 방법

1. 입력칸에 할일을 입력합니다.
2. 아래 할일 목록에서 입력된 할일들과 완료 체크박스를 확인할 수 있습니다.
3. 완료한 할일은 체크박스를 눌러 삭제합니다.

## 설치 방법

1. zip으로 압축을 풀고 todolist.html을 브라우저로 엽니다.
